FAQ

Common questions

How do I import an existing collection?

Settings → Import → pick Goodreads (books), Letterboxd (movies), or Discogs (vinyl), then pick the CSV file. Each format expects the exporter's stock columns — nothing to configure.

Does barcode scanning need internet?

Yes — barcode scans send the digits to Google's Gemini 3.5 Flash for a deep lookup that returns title, manufacturer, year, and price range. Scanning still works offline, but you'll get the barcode only, not the title or cover.

Does AI photo identification need internet?

Yes — Kabynyt sends the photo to Google's Gemini 3.5 Flash for identification. You can supply your own Gemini API key in Settings, or use the bundled key (limited to 5 free identifications, then $0.99/month).

How do I back up my collection?

iCloud sync is automatic if enabled. Settings → Backup & Restore also exports a JSON snapshot you can save anywhere.

Is Kabynyt free?

The app is free to download and use indefinitely. Smart Add — the AI-powered photo identification and barcode deep-lookup — is free for the first five identifications, then $0.99 a month if you'd like more. Everything you've already added stays yours either way.

I deleted something by accident

Settings → Recently Deleted. Items live there for 30 days before they're purged.
